Sistemas Operativos

Tema 1.1
¿Que es un sistema Operativo?

Un Sistema Operativo es un programa que actúa como intermediario entre el usuario y el Hardware de un computador. Su propósito es proporcionar un entorno en el cual el usuario pueda
programas de manera cómoda y eficiente.

Máquina Extendida

El Sistema Operativo ofrece a los programas un conjunto de servicios, o “llamadas al sistema”, que pueden solicitar cuando lo necesiten,
una visión de maquina amplia.

Máquina Virtual

Separan totalmente las funciones de multiprogramación y maquina extendida. Estas crean copias “exactas del Hardware simple con los
componentes que posee una maquina real. Pueden ejecutar cualquier Sistema Operativo que se ejecute de manera directa sobre el Hardware.

Dispositivos de Entrada/Salida

Pueden intercambiar datos directamente con el procesador. El procesador puede iniciar una lectura o escritura en la memoria, o leer datos de un módulo I/O y escribir en dicho módulo.

User Interface

Es el modulo del sistema operativo que permite la comunicación de manera interactiva con el sistema, es el intérprete de los mandatos o Shell.

Administrador del Sistema

El conjunto de instrucciones ampliadas que proporciona el Sistema Operativo y son las llamadas al sistema., eliminan y utilizan objetos del software controlados por el Sistema Operativo (procesos y archivos).
